# Job Application Pipeline Tracker

A comprehensive tracking system for managing active job applications, phone screens, interviews, and job search progress.

## Contents

- **JOB_APPLICATION_PIPELINE_MASTER.md** — Main tracking document with:
  - Pipeline summary by status (applications, phone screens, interviews)
  - Detailed information on active opportunities
  - Interview tracking with notes and outcomes
  - Application history and timeline

## Features

- Color-coded status tracking (🔵 interviews, 📞 phone screens, 🟡 advocacy, etc.)
- Salary ranges and job fit percentages
- Interview performance notes and confidence levels
- Referral tracking and contact information
- Timeline and decision dates

## Usage

Update the master file as you progress through the job search pipeline. Each status update is tracked with dates and relevant notes for future reference.
